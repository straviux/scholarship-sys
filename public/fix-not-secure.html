🔒 Fix "Not Secure" Warning

Quick & Easy Guide to Install Security Certificate

❌ What's the Problem?

Your browser shows "Not Secure" or "Your connection is not private" when accessing this system because the security certificate is not installed on your computer. This is normal for internal systems and can be fixed in 2 minutes!

✅ The Solution (Choose One Method)

🚀 Method 1: One-Click Install (EASIEST!)

📥 Download Automatic Installer
Download the Installer

Click the green button above. The file "install-certificate.bat" will download to your Downloads folder.

Run the Installer

Open your Downloads folder and double-click "install-certificate.bat"

📁 Look for: install-certificate.bat in Downloads folder
Click "Yes" on Windows Prompt

Windows will ask: "Do you want to allow this app to make changes?" → Click YES

🛡️ Windows Security: Click [Yes]
Wait for Success Message

The installer will show "Installation Complete!" - This means it worked!

Restart Your Browser

IMPORTANT: Close ALL browser windows (not just tabs) and reopen your browser

🎉 Done!

Now access the system normally - No more "Not Secure" warnings!

📋 Method 2: Manual Installation

📄 Download Certificate File
Download the Certificate

Click the blue button above to download "rootCA.pem"

Double-Click the Certificate File

Find "rootCA.pem" in your Downloads folder and double-click it

Click "Install Certificate"

A window will open showing certificate details. Click the "Install Certificate..." button

Select "Current User"

Choose "Current User" and click "Next"

Select Certificate Store

Choose "Place all certificates in the following store" → Click "Browse"

Choose "Trusted Root Certification Authorities"

Select this exact folder from the list → Click "OK"

Complete the Wizard

Click "Next" → Click "Finish" → Click "Yes" on the security warning

Restart Your Browser

Close ALL browser windows and reopen

⚠️ Still Not Working?

1. Make sure you restarted your browser COMPLETELY - Close all windows, don't just close tabs.
2. Clear your browser cache: Press Ctrl+Shift+Delete → Check "Cached images and files" → Clear
3. Check certificate installation: Press Windows+R → Type "certmgr.msc" → Press Enter → Look for certificate in "Trusted Root Certification Authorities"
4. Still stuck? Contact IT support with this error message.
